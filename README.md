# tsp-trustbadge

TrustBadge component scaffold for the clean vNext Trust Standard Protocol
architecture.

## Doctrine

TSP verifies evidence state, not truth or final legal outcome.

## Scope

TrustBadge is a visible inspection surface for TSP evidence. It should help a
person open or inspect a receipt, verifier result, or evidence view.

The component does not verify by itself, does not grant production issuance
rights, does not grant TrustBadge authorization, and does not grant Official TSP
Conformance Attestation.
